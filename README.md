# LaTeX-Workshop

A series of workshops to learn how to make :fire: documents using LaTeX :green_book:

   <pre>An introduction to the typesetting tool</pre>
    - [Let's Code It!](https://github.com/humdrumcomet/LaTeXCodingSession)
